Basic Probability Formula

For a 5x5 grid (25 tiles) with M mines:

P(revealing N gems) = [(25-M)! / (25-M-N)!] / [25! / (25-N)!]

In simpler terms: multiply the probability of each successful reveal.

JavaScript Probability Calculator

function calculateProbability(mines, gems) {
  const totalTiles = 25;
  const safeTiles = totalTiles - mines;

  let probability = 1;
  for (let i = 0; i < gems; i++) {
    probability *= (safeTiles - i) / (totalTiles - i);
  }

  return probability;
}

// Example: 3 mines, 5 gems
console.log(calculateProbability(3, 5)); // 0.5038...
